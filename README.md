# parse_primary_json_data.py

Модуль для парсинга данных из Yandex-форм в структурированный JSON-формат.

## Назначение

Преобразует сырые данные опросов, собранных через Yandex Forms, в удобный для дальнейшего анализа и визуализации JSON-формат с группировкой ответов по категориям.

## Выходной формат

Функция `read_base_dict_from_json` возвращает JSON-объект со следующими разделами:

| Поле | Описание |
|------|----------|
| `age` | Распределение по возрастным группам |
| `place_of_residence` | Распределение по месту жительства |
| `profession` | Распределение по роду деятельности |
| `frequency_of_using_bypass` | Частота использования средств обхода блокировок |
| `blocked_services_using` | Использование заблокированных сервисов |
| `political_activity_dynamic` | Динамика изменения политических взглядов |
| `technical_skills_dynamic` | Динамика изменения технических навыков |
| `blocks_efficiency_opinion` | Мнение об эффективности блокировок |
| `publish_opinion_in_media` | Практика публикации мнения в соцсетях |

## Пример выходных данных

```json
{
    "age": {
        "less 18": 57,
        "18-25": 38,
        "26-40": 9,
        "41-60": 22,
        "60 and elder": 10
    },
    "place_of_residence": {
        "Moscow": 98,
        "Moscow area": 15,
        "Saint-Petersburg": 0,
        "another million citizens town": 1,
        "another less than million citizens town": 21,
        "settlement or village": 1
    },
    "profession": {
        "schoolboy": 55,
        "technical student": 20,
        "humanities student": 12,
        "IT-sector worker": 8,
        "not IT-sector worker": 13,
        "entrepreneur": 1,
        "budgetary sphere": 10,
        "pensioner": 8,
        "another": 9
    }
}
```

# statistics_analyzer.py

Модуль для статистического анализа данных опроса об интернет-блокировках.

## Назначение

Модуль позволяет анализировать данные опроса, группируя респондентов по различным социально-демографическим признакам (возраст, место жительства, профессия и т.д.) и вычислять процентное распределение ответов внутри каждой группы.

## Основные функции

### `get_raw_group_analysis(survey_data_filename)`

Вызывает `get_data_for_special_group`.
Возвращает сырые (необработанные) данные по выбранной пользователем группе респондентов в абсолютных числах.

**Параметры:**
- `survey_data_filename` (`str`) — путь к JSON-файлу с данными опроса

**Возвращает:** `dict` — словарь с абсолютными значениями (количеством респондентов)

### `get_processed_group_analysis(survey_data_filename)`

Вызывает `get_statistics_for_special_group`.
Возвращает обработанные данные по выбранной пользователем группе респондентов в процентном выражении.

**Параметры:**
- `survey_data_filename` (`str`) — путь к JSON-файлу с данными опроса

**Возвращает:** `dict` — словарь с процентными значениями (строки вида "15.30%")

### `get_data_for_special_group(group_index, survey_data_filename)`

Собирает сырые данные по выбранному признаку группировки.

**Параметры:**
- `group_index` (`int`) — индекс признака группировки (0-8)
- `survey_data_filename` (`str`) — путь к JSON-файлу с данными опроса

Индекс признака группировки определяется по следующим соответствиям:
0. Возраст
1. Место жительства
2. Род деятельности
3. Обход блокировок
4. Заблокированные сервисы
5. Динамика изменения политических взглядов
6. Динамика изменения уровня технических навыков
7. Субъективная оценка эффективности блокировок
8. Отношение к публикации своего мнения в медиа

**Возвращает:** `dict` — словарь вида:
```python
{
    "Moscow": {
        "age": {"less 18": 5, "18-25": 10, ...},
        "place_of_residence": {...},
        "profession": {...},
        ...
    },
    "Moscow area": {...},
    ...
}
```

Пример выше показывает, что разделение респондентов происходит по их месту жительства.
Так, среди живущих в `Москве`:
- 5 человек младше 18 лет
- 10 человек в возрасте от 18 до 25 лет
- и т.д.

### `get_statistics_for_special_group(group_index, survey_data_filename)`

**Главная функция модуля.** Выполняет полный анализ данных по выбранному признаку группировки и преобразует абсолютные значения в проценты.

**Параметры:**
- `group_index` (`int`) — индекс признака группировки (0-8)
- `survey_data_filename` (`str`) — путь к JSON-файлу с данными опроса

**Возвращает:** `dict` — словарь с процентными значениями

**Пример результата:**
```python
{
    "Moscow": {
        "age": {"less 18": "15.30%", "18-25": "42.80%", ...},
        "profession": {"schoolboy": "5.10%", ...},
        "respondents_quantity": 98
    },
    "Moscow area": {
        "age": {"less 18": "10.00%", ...},
        "respondents_quantity": 15
    }
}
```

Пример выше показывает, что от живущих в `Москве` респондентов несовершеннолетние
составляют 15.30%, молодые люди от 18 до 25 лет - 42.80%, а всего живущих 
в `Москве` - 98 человек. Аналогично для 15-и человек, живущих в `Московской области` 
несовершеннолетние составляют 10.00%.
