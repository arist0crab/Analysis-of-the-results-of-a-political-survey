# parse_primary_json_data.py

Модуль для парсинга данных из Yandex-форм в структурированный JSON-формат.

## Назначение

Преобразует сырые данные опросов, собранных через Yandex Forms, в удобный для дальнейшего анализа и визуализации JSON-формат с группировкой ответов по категориям.

## Выходной формат

Функция `read_base_dict_from_json` возвращает JSON-объект со следующими разделами:

| Поле | Описание |
|------|----------|
| `age` | Распределение по возрастным группам |
| `place_of_residence` | Распределение по месту жительства |
| `profession` | Распределение по роду деятельности |
| `frequency_of_using_bypass` | Частота использования средств обхода блокировок |
| `blocked_services_using` | Использование заблокированных сервисов |
| `political_activity_dynamic` | Динамика изменения политических взглядов |
| `technical_skills_dynamic` | Динамика изменения технических навыков |
| `blocks_efficiency_opinion` | Мнение об эффективности блокировок |
| `publish_opinion_in_media` | Практика публикации мнения в соцсетях |

## Пример выходных данных

```json
{
    "age": {
        "less 18": 57,
        "18-25": 38,

    }
}
```

# statistics_analyzer.py

Модуль для получения статистических данных из опроса об интернет-блокировках.

## Две главные функции

### 1. `get_raw_group_analysis(survey_data_filename)`

**Что делает:** Возвращает сырые (необработанные) данные опроса в абсолютных числах (количество человек).

**Когда использовать:** Когда вам нужно узнать точное количество респондентов, выбравших тот или иной вариант ответа.

**Как использовать:**

```python
from statistics_analyzer import get_raw_group_analysis

# Передайте имя файла с данными опроса
result = get_raw_group_analysis("survey_data.json")

# В консоли появится меню. Выберите цифру от 0 до 9.
# Например, введите 0 — получите данные по возрасту.
```

**Что вы получите:**

```python
{
    "Moscow": {
        "age": {"less 18": 5, "18-25": 42, "26-40": 15},
        "profession": {"schoolboy": 2, "technical student": 15},
        ...
    },
    "Moscow area": {
        "age": {"less 18": 2, "18-25": 8, "26-40": 5},
        ...
    }
}
```

Все значения — целые числа (количество человек).

---

### 2. `get_processed_group_analysis(survey_data_filename)`

**Что делает:** Возвращает обработанные данные в процентах. Также позволяет выбрать конкретную подгруппу (например, только "Москву" или только "Санкт-Петербург").

**Когда использовать:** Когда вам нужны проценты для диаграмм, инфографики или сравнения долей.

**Как использовать:**

```python
from statistics_analyzer import get_processed_group_analysis

# Передайте имя файла с данными опроса
result = get_processed_group_analysis("survey_data.json")
```

**Пошаговая инструкция:**

1. **Первый выбор** — в консоли появится меню с признаками (возраст, место жительства и т.д.). Введите цифру от 0 до 8.

2. **Второй выбор** — появится подменю со списком подгрупп. Например, если вы выбрали "Место жительства", увидите:
   - Введите номер конкретной подгруппы (например, `0` — получите данные только по Москве)
   - Или введите последний номер (например, `6`) — получите данные по всем подгруппам сразу

**Что вы получите:**

Если при выводе первого меню вы выбрали признак `Место жительства`, а при выводе второго
из возможных вариантов была выбрана `Москва`, то полученные данные будут иллюстрировать
процентное отношение по категориям **только среди москвичей**:

```python
{
    "Moscow": {
        "age": {"less 18": "5.10%", "18-25": "42.86%", "26-40": "15.31%"},
        "profession": {"schoolboy": "2.04%", "technical student": "15.31%"},
        "respondents_quantity": 98
    }
}
```

В примере выше рассматриваются респонденты только из Москвы, среди которых:
- 5.10% младше 18 лет
- 42.86% в возрасте от 18 до 25 лет
- 15.31% в возрасте от 26 до 40 лет
- 2.04% школьников
- 15.31% составляют студенты технической специальности
- всего респондентов из Москвы 98

Если выбрали "Вывести все":
```python
{
    "Moscow": {
        "age": {"less 18": "5.10%", "18-25": "42.86%"},
        "respondents_quantity": 98
    },
    "Moscow area": {
        "age": {"less 18": "13.33%", "18-25": "53.33%"},
        "respondents_quantity": 15
    }
    ...
}
```

**Важно:** В результатах появилось поле `respondents_quantity` — общее количество респондентов в данной группе.

---

## Сравнение двух функций

| Что нужно получить | Какую функцию вызывать | Формат результата |
|---|---|---|
| Точное количество человек | `get_raw_group_analysis()` | Целые числа |
| Проценты для диаграмм | `get_processed_group_analysis()` | Строки с `%` |
| Данные по конкретному городу/возрасту | `get_processed_group_analysis()` | Выбрать подгруппу во втором меню |
| Данные по всем подгруппам сразу | `get_processed_group_analysis()` | В конце подменю выбрать "Вывести все" |

---

## Пример

Допустим, вы делаете круговую диаграмму "Распределение респондентов по возрасту в Москве".

```python
from statistics_analyzer import get_processed_group_analysis

# Получаем данные
data = get_processed_group_analysis("survey_data.json")

# В консоли:
# 1. Выбираете "0. Возраст"
# 2. Выбираете "0. Москва"

# В переменной data окажется:
# {"Moscow": {"age": {"less 18": "5.10%", "18-25": "42.86%", ...}}}
```

# visualization_core.py 

Модуль для создания инфографики на основе результатов исследований.

## Основные возможности

- **Donut Chart**: круговая диаграмма.

## Структура и зависимости

Модуль является частью системы анализа опросов и ожидает следующую структуру проекта:

```text
project_root/
├── config.py               # Глобальные пути и палитра
├── main.py                 # Точка входа
├── errors.py
└── visualization/
    └── vizualizator_core.py # Данный модуль
└── data_processing/
    ├── parse_primary_json_data.py
    ├── social_group_analysis.py
    └── translaters.py
```
## Основная точка входа — функция plot_overall_statistics

Она берет данные из JSON, обрабатывает их и генерирует два изображения - круговую диаграмму и легенду к ней.

## Пример использования:
```python
from visualization.vizualizator_core import plot_overall_statistics

# Параметры: путь к JSON, ключ категории, имя для файлов
plot_overall_statistics("results.json", "political_activity_dynamic", "Political_Views")
```

## Результат работы
После выполнения в папке, указанной в PLOTS_DIR, появятся файлы:

- Political_Views.png — диаграмма с числовыми индексами и процентами.

- Political_Views_legend.png — текстовая расшифровка индексов.
